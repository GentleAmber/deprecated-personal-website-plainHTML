A site where users can log in to their own accounts & keep track of their book notes
This website uses PostgreSQL to save user data, and Open Library Covers API to fetch covers for the books whose serial numbers are provided.
Users can write notes, give ratings, etc.

While every effort has been made to support different devices, this website is best experienced on a desktop computer.
